Google Calendar Integration

Auto-create birthday events when employees are added. Requires a Google service account.

📋 Calendar ID —

📖 Setup Guide (click to expand)

Step 1: Go to Google Cloud Console → Create or select a project.

Step 2: Enable Google Calendar API from the API Library.

Step 3: Go to Credentials → Create Credentials → Service Account.

Step 4: Name it (e.g. employee-management-calendar) and click Done.

Step 5: Click the service account → Keys → Add Key → JSON → Download the file.

Step 6: Save the JSON file to your project folder (e.g. service-account-key.json).

Step 7: Open Google Calendar → Settings → Add Calendar → Create new calendar (optional).

Step 8: Go to calendar settings → Share with specific people → Add the service account email (from the JSON key file under client_email) with Make changes to events permission.

Step 9: Enter the JSON file path and Calendar ID above, then click Save Calendar Config.
